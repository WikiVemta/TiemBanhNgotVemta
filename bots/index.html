Bot hiện đang hoạt động trong server



Dyno

  • Dyno là một bot quản lý Discord với nhiều tính năng: tự động quản lý server, cảnh báo, chống spam, quản lý vai trò, tạo lệnh tùy chỉnh,...
  • Được phát triển để giúp admin/server owner dễ kiểm soát server mà không cần nhiều thao tác thủ công.
Mời bot


Mimu

  • Mimu là bot hỗ trợ server với tính năng nổi bật như: ửi tin nhắn chào mừng/tạm biệt, autoresponder.
Mời bot


Birthday Bot

  • Birthday Bot là bot sẽ cho phép các thành viên trong server chia sẻ ngày sinh và thông báo ngày sinh người khác.
Mời bot


Carl Bot

  • Carl Bot là bot hỗ trợ server với tính năng nổi bật như: quản lý server, tự động gán role khi vào server, reaction role,...
  • Được phát triển để giúp admin/server owner dễ kiểm soát server mà không cần nhiều thao tác thủ công.
Mời bot


Arcane

  • Arcane là bot hỗ trợ server với tính năng nổi bật như: hệ thống level & XP.
Mời bot


OWO
